# Hotel-Booking-and-Management-System
This is a database project where I designed a hotel booking and management system to organize guest, room, reservation, employee, service, and payment data.

I worked on reducing data redundancy, improving storage accuracy, supporting fast querying using SQL, implementing Extensible Entity Relationship Modeling (EER), and designing relational diagrams. I also defined the properties of the guest and employee entities, implemented basic SQL queries (SELECT, FROM, WHERE), and contributed to designing and converting the EER diagram into a relational diagram.
